## 2. Create the Facebook Page

1. [facebook.com/pages/create](https://www.facebook.com/pages/create)  
2. Category: **Advertising / marketing** or **Consulting agency** (pick closest).  
3. Name: e.g. **369 Leads** or **369 Leads — KW Roofing Lead Gen**.  
4. Upload **profile** + **cover** from `assets/`.  
5. **About:** 2–3 sentences — exclusive Meta leads for **KW / Cambridge / Guelph** roofers; link to your live site or Carrd.  
6. **CTA button:** **Book now** → your **Calendly** link.

## 3. Meta Business Manager

1. [business.facebook.com](https://business.facebook.com) → create **Business portfolio**.  
2. **Ad account:** Create ad account inside BM; add **payment method**.  
3. **People:** Add your personal FB profile as Admin.  
4. **Pages:** Connect your new Page to the business.  

You **do not** need to run ads on day one to finish this—just have BM + ad account **ready** so the first client or proof campaign isn’t blocked.

## 4. After website is on a real domain

- Put that URL in Page **website** field + LinkedIn Featured + posts.  
- Optionally install **Meta Pixel** on the agency site (requires a short script in `<head>`—you or a dev).

## 5. Your Launch Guide “next steps” summary

- Spend **foundation** dollars on Canva / domain / Carrd per your doc; your **full agency site** can remain this `index.html` project once hosted.  
- **Outbound** can start as soon as **LinkedIn + Calendly + Page** exist; site URL can be “coming soon” for 24h if needed.  
- Hold **ad test** budget until client pay or deliberate proof campaign.
